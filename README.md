# Bookworming to Dinner

This is a 2D platformer made in Unity. Play as a bookworm and reach the top of the shelf for the yummy antique books! Collect page crumbs, worm coins, and maybe say hi to the dust bunnies?

# Getting Started

## Running the Game
* Play the game in browser at itch.io: https://arcane-worm.itch.io/bookworming-to-dinner 

## Installing and Running in Unity
* Clone this repository from GitHub: [https://github.com/ArcaneWorm/Microplastics-Data-Analysis.git](https://github.com/ArcaneWorm/Platformer-Demo.git)
* Ensure you have Unity version 2022.3.50f installed
* Select the MainMenu scene or Bookworming scene and press play!
